# vsc-test-suite
VSC test suite

## How to run the tests

run script that works recursively

```
cd vsc-test-suite
./run.sh args
```

Optional `args` will be passed to the reframe command.

## Output location

Log files and output will be saved in $HOME/reframe

## Requirements 

- Reframe 3.10.1 installed as a module
- Python3
